# 在线考试管理系统 - 项目简介

## 项目概述

在线考试管理系统是一个基于SpringBoot和Vue.js的前后端分离教育平台，旨在为教育机构提供高效、智能的数字化考试解决方案。系统支持管理员、教师和学生三种角色，实现了从考试创建、在线答题到成绩分析的全流程管理。

## 技术架构

- **前端框架**: Vue 3 + Element Plus
- **后端框架**: SpringBoot
- **数据库**: MySQL
- **架构模式**: 前后端分离 + 微服务架构
- **构建工具**: Vite + Maven

## 核心功能模块

### 1. 管理员端功能

- **系统概览**: 数据统计卡片展示用户数、试卷数、公告数及系统活跃度
- **用户管理**: 对学生、教师账号进行增删改查和状态管理
- **试卷管理**: 全平台试卷的创建、审核与维护
- **公告管理**: 系统级公告的发布与权限控制
- **系统监控**: 操作日志记录与系统性能监控

### 2. 教师端功能

- **课程管理**: 创建和管理教学课程，关联学生与试卷
- **试卷管理**: 智能组卷、题型配置和考试参数设置
- **在线监考**: 实时监控考试过程，防作弊检测
- **成绩批改**: 客观题自动评分，主观题在线批阅
- **成绩分析**: 学生成绩统计与学习情况分析

### 3. 学生端功能

- **考试中心**: 查看可参加考试，支持科目和状态筛选
- **在线考试**: 友好的考试界面，支持多种题型作答
- **成绩查询**: 个人成绩查看与成绩趋势分析
- **公告查看**: 接收学校和教师发布的各类通知

## 系统特色

### 技术亮点

- **响应式设计**: 基于Element Plus的现代化UI组件
- **模块化架构**: 清晰的前后端分离，便于维护和扩展
- **安全性保障**: 完善的权限控制和数据验证机制
- **高性能设计**: 支持高并发考试场景

### 教育特色

- **多题型支持**: 选择题、填空题、主观题等多种题型
- **灵活组卷**: 支持随机组卷和固定组卷两种模式
- **智能防作弊**: 切屏检测、时间监控等防作弊机制
- **数据分析**: 可视化成绩统计和学习进度跟踪
